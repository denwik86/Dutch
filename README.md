# 🇳🇱 Dutch A2 Coach — Инструкция по установке
## От нуля до работающего бота за 15 минут

---

## ШАГ 1: Создай Telegram-бота (2 минуты)

1. Открой Telegram, найди **@BotFather** (синяя галочка)
2. Напиши: `/newbot`
3. Введи имя бота: `Dutch A2 Coach` (или любое)
4. Введи username: `dutch_a2_viktar_bot` (должен заканчиваться на `bot`)
5. BotFather пришлёт **токен** вида `1234567890:AAHfiqks...`
   → Скопируй его, это `TELEGRAM_TOKEN`

---

## ШАГ 2: Узнай свой Telegram Chat ID (1 минута)

1. Напиши боту **@userinfobot** в Telegram
2. Он ответит твой числовой ID, например `394821047`
   → Это `CHAT_ID`

---

## ШАГ 3: Получи Anthropic API Key (3 минуты)

1. Открой https://console.anthropic.com
2. Войди или зарегистрируйся
3. Слева → **API Keys** → **+ Create Key**
4. Скопируй ключ вида `sk-ant-api03-...`
   → Это `ANTHROPIC_KEY`

> 💰 Стоимость: ~$0.50–2.00 в месяц при ежедневном использовании

---

## ШАГ 4: Запуск (выбери один из вариантов)

### Вариант A: Railway.app (рекомендуется, бесплатно)

1. Зайди на https://railway.app → Sign up with GitHub
2. New Project → Deploy from GitHub repo
   - Сначала загрузи файлы на GitHub (create repository → upload files)
3. В Railway → Variables → добавь три переменные:
   ```
   TELEGRAM_TOKEN = твой_токен
   ANTHROPIC_KEY  = твой_ключ  
   CHAT_ID        = твой_id
   ```
4. Deploy → бот запустится автоматически
5. Готово! Бот работает 24/7

---

### Вариант B: Локально на своём компьютере

**Требования:** Python 3.10+

```bash
# 1. Создай папку и перейди в неё
mkdir dutch_coach && cd dutch_coach

# 2. Скопируй файлы bot.py и requirements.txt в эту папку

# 3. Создай виртуальное окружение
python -m venv venv
source venv/bin/activate   # Mac/Linux
# или: venv\Scripts\activate  # Windows

# 4. Установи зависимости
pip install -r requirements.txt

# 5. Создай файл .env (скопируй из .env.example и заполни)
cp .env.example .env
# Открой .env в текстовом редакторе и вставь свои токены

# 6. Добавь в bot.py загрузку .env (первые строки после импортов):
# from dotenv import load_dotenv
# load_dotenv()

# 7. Запуск
python bot.py
```

**Чтобы бот работал когда ты закрыл компьютер** — оставь процесс запущенным
или используй `screen`/`tmux` на Mac/Linux.

---

### Вариант C: Render.com (бесплатно, альтернатива Railway)

1. https://render.com → New → Web Service
2. Connect GitHub repository
3. Build Command: `pip install -r requirements.txt`
4. Start Command: `python bot.py`
5. Environment Variables: добавь `TELEGRAM_TOKEN`, `ANTHROPIC_KEY`, `CHAT_ID`

## ШАГ 5: Проверка (1 минута)

1. Найди своего бота в Telegram (по username из шага 1)
2. Напиши `/start`
3. Бот должен ответить приветствием на русском

## Структура обучения (16 недель)

### Фаза 1: Основы (нед 1–4, апр–май)
- Алфавит и произношение
- 200 базовых слов A2
- Знакомство с форматом экзамена
- Первые 2 темы KNM: Wonen + Zorg

### Фаза 2: Практика (нед 5–9, май–июнь)
- Ежедневные тексты с NOS.nl
- Аудио: диалоги и объявления
- Все 8 тем KNM
- Словарь до 800 слов

### Фаза 3: Имитация (нед 10–15, июль–нач авг)
- Еженедельный mock-exam (/mock по субботам)
- Скоростное чтение
- Повтор слабых тем
- Словарь 1000–1200 слов

### Фаза 4: Финальная (нед 16)
- Только лёгкое повторение
- Отдых накануне
- Уверенность и спокойствие
